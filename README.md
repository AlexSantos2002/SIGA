# SIGA

## Objetivo
Sistema integrado de gestão animal é uma plataforma intuitiva com o objetivo de centralizar os dados de organizações e promover uma adoção e gestão de animais mais responsavel. Esta plataforma tem diversas funções, desde a gestão de animais (registo, cuidados especiais como vacinação e medicação etc), gestão de adotantes (informações, se tem "flags" que maus comportamentos em adoções anteriores etc, dados estatisticos relativos a cada organização registada).

# Tecnolgias usadas

- Angular
- Supabase (Postgresql)
- Typescript
- Html 
- Css

## Outras ferramentas utilizadas

- Extensão prettier do VSCODE para padronização da formatação em todo o projeto
